# 繁体字中国語 頻出単語リスト（発音・日本語訳付き）

本リポジトリでは、台湾の標準的な繁体字中国語の学習用データ（TSV形式）を公開しています。

発音特訓アプリ https://github.com/hiansit/pronunciation_trainer_v1

のインポートデータ等として利用可能です。

## データ仕様
* 第1列：繁体字単語
* 第2列：日本語訳
* 第3列：発音記号（注音符号 / 台湾拼音）

## 資料來源（データの出所について）
本データセットに含まれる「単語リスト」および「出現順位（頻度）」は、中華民国教育部が過去に公開した以下の資料をベースに抽出しています。

* **元データ**: 教育部『國語辭典簡編本編輯資料字詞頻統計報告』（民国86年）の「詞頻總表」 
https://language.moe.gov.tw/001/Upload/files/SITE_CONTENT/M0001/PIN/biau2.htm
* **提供元**: 中華民国教育部 語文成果入口網
https://language.moe.gov.tw/index

## 資料處理與免責聲明（加工のプロセスと免責事項）
公開にあたり、元データに対して以下の独自加工を施しています。

1. **抽出**: 元データのテキストファイルから、頻度順に単語を抽出・クレンジング。
2. **翻訳と発音付与**: 大規模言語モデル（AI）を利用し、抽出した単語に対して日本語訳と台湾標準の発音記号（注音・ピンイン）を独自に生成して付加。

**【重要】**
本リポジトリのデータ（日本語訳、発音記号の正確性、および加工後のデータ構造）は作成者が独自に生成・編纂したものであり、**台湾教育部の公式な見解や品質保証を受けたものではありません。** 本データの利用により生じたいかなる損害についても、データ提供元（教育部）および作成者は責任を負いません。
